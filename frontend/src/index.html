Bem vindo ao sistema de Estacionamento WEB!

Ao realizar login, você terá acesso ao dropdown de opções, onde poderá registrar entrada e saída de veículos, cadastro de cliente, revisão de dados financeiros e realizar logout

Esta aplicação foi desenvolvida com propósitos avaliativos e didáticos.

A aplicação foi desenvolvida com:

  • Front-End
    • JavaScript Vanilla (sem uso de frameworks)
    • HTML
    • CSS
  • Back-End
    • NodeJS
  • Banco de Dados
    • MySQL WorkBench

O autor não possuia experiência com JavaScript sem utilização de frameworks e nem experiência em back-end com NodeJS.

Dificuldades e aspectos a serem melhorados incluem: Modularização nos arquivos de back-end, modularização de componentes nos arquivos de front-end, disponibilização de banco de dados na web, relacionamento entre clientes e veículos, validações de campos e mensagens
